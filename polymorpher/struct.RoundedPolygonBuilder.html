Expand description

Provides a more convenient and familiar way to create certain shapes using builder pattern, helping to avoid functions with hellish Option<T> arguments.

Currently supports the creation of the following shapes:

  • Rectangles
  • Circles
  • Stars
  • Pills
  • Pill stars
